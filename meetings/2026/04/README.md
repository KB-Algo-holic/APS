# 1️⃣ 강의

## 📌 강의 주제

### Stack & Queue & Deque

---

## ❓ Why?

### 1) 코딩테스트 핵심 자료구조

다음 개념의 이해를 위해 반드시 필요합니다:

- DFS → Stack 기반
- BFS → Queue 기반
- Sliding Window / 0-1 BFS → Deque 기반

즉,

```
DFS 이해 = Stack 이해
BFS 이해 = Queue 이해
최적화 BFS 이해 = Deque 이해
```

---

### 2) 지난주 과제 활용 미흡

지난 과제에서 기대했던 접근 방식:
