# 1️⃣ 강의

## 📌 강의 주제

### Graph & DFS & BFS

---

## ❓ Why?

### 1) 코딩테스트 핵심 탐색 알고리즘

다음 유형을 해결하기 위해 반드시 필요합니다:

- 최단 경로 탐색
- 연결 요소 탐색
- 완전 탐색
- 시뮬레이션 기반 탐색

즉,

```text
탐색 문제 이해 = Graph 이해
깊이 우선 탐색 = DFS 이해
최단 거리 탐색 = BFS 이해
```

---

### 2) 실전 문제 빈출 유형

최근 코딩테스트에서 매우 높은 빈도로 등장하는 유형:

- 미로 탐색
- 네트워크 연결
- 섬 개수 구하기
- 경로 탐색

대표 예시:

- [게임 맵 최단거리](https://school.programmers.co.kr/learn/courses/30/lessons/1844)
- [타겟 넘버](https://school.programmers.co.kr/learn/courses/30/lessons/43165)

탐색 방식을 선택하는 것만으로도 풀이 난이도가 크게 달라질 수 있음

예시:

```text
모든 경우 탐색 → DFS
최단 거리 탐색 → BFS
```
