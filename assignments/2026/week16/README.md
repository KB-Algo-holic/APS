# 5월 4주차 - 누적합 (Prefix Sum)

## 목표 🎯

누적합(Prefix Sum)은

```
구간 합을 매우 빠르게 계산하는 알고리즘
```

입니다.

대표 사용 목적

- 구간 합 계산
- 특정 범위 합 계산
- 시간복잡도 최적화
- 2차원 영역 합 계산

특히 다음 유형에서 자주 등장합니다

- 부분합 문제
- 배열 합 문제
- 구간 합 문제
- 행렬 영역 합 문제

이번 주차 목표

✔ 누적합 개념 이해  
✔ 1차원 누적합 구현 가능  
✔ 2차원 누적합 이해  
✔ 구간합 문제 해결 가능  
✔ Sliding Window와 차이 이해

---

# 누적합(Prefix Sum)이란

누적합은

```
앞에서부터 합을 미리 저장해두는 방식
```

입니다.

예시 배열

```text
[1, 2, 3, 4, 5]
```

누적합 배열

```text
[1, 3, 6, 10, 15]
```

의미

```text
index 0까지 합 → 1
index 1까지 합 → 3
index 2까지 합 → 6
```

# 왜 필요한가 🤔

문제

구간 합을 여러 번 구해야 함

예시

```text

3~7 구간 합

```

```text

```

```text
O(NQ)
```

비효율적

# 구간 합 공식 ⭐

L ~ R 합

```text
prefix[R] - prefix[L - 1]
```

예시

2 ~ 4 합

```text
10 - 1 = 9
```

결과

```text
2 + 3 + 4 = 9
```

---

# 1차원 누적합 구현

## 배열 선언

```java
int[] prefix = new int[n + 1];
```

---

## 누적합 계산

```java
for (int i = 1; i <= n; i++) {

    prefix[i] =
        prefix[i - 1] + arr[i];

}
```

---

## 구간합 계산

L ~ R

```java
int sum =
    prefix[r] - prefix[l - 1];
```

시간복잡도

```text
O(1)
```

---

# 예제 코드

```java
int[] arr = {0, 1, 2, 3, 4, 5};

int[] prefix = new int[arr.length];

for (int i = 1; i < arr.length; i++) {

    prefix[i] =
        prefix[i - 1] + arr[i];

}

int left = 2;
int right = 4;

int answer =
    prefix[right] -
    prefix[left - 1];
```

# 2차원 누적합 📊

행렬 문제 핵심

대표 문제

```text
직사각형 영역 합
```

예시

```text
1 2 3
4 5 6
7 8 9
```

---

# 2차원 누적합 공식

```text
dp[i][j]
=
현재값
+ 위쪽
+ 왼쪽
- 겹치는 영역
```

공식

```text
dp[i][j]
=
board[i][j]
+ dp[i-1][j]
+ dp[i][j-1]
- dp[i-1][j-1]
```

---

# 2차원 누적합 구현

```java
for (int i = 1; i <= n; i++) {

    for (int j = 1; j <= m; j++) {

        dp[i][j] =
            board[i][j]
            + dp[i - 1][j]
            + dp[i][j - 1]
            - dp[i - 1][j - 1];

    }

}
```

---

# 직사각형 구간 합 공식 ⭐

```text
(x1, y1) ~ (x2, y2)
```

공식

```text
dp[x2][y2]
- dp[x1 - 1][y2]
- dp[x2][y1 - 1]
+ dp[x1 - 1][y1 - 1]
```

# 누적합 vs Sliding Window

헷갈리는 포인트

| 항목 | 누적합 | 슬라이딩 윈도우 |
|------|--------|----------------|
| 구간 길이 | 자유 | 고정 |
| 합 계산 | O(1) | O(1) |
| 목적 | 여러 구간 | 연속 구간 |
