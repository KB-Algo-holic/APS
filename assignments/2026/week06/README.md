# 3월 2주차 - 문자열 (String)

## 목표
문자열 처리 문제를 통해 **문자 단위 조작과 패턴 분석 능력**을 기릅니다.

문자열 문제는 알고리즘 테스트에서 **매우 자주 등장하는 기본 유형**입니다.

---

## 주요 문제 유형

### 1️⃣ 문자 탐색

문자열을 한 글자씩 확인하는 문제

예시
- 특정 문자 개수 세기
- 문자 변경

패턴
for (char c : str.toCharArray())

---

### 2️⃣ 문자열 변환

문자열을 변형하는 문제

예시
- 대소문자 변환
- 문자열 뒤집기
- 특정 문자 제거

---

### 3️⃣ 문자열 비교

문자열 패턴 비교

예시
- 팰린드롬
- 접두사 / 접미사
- 문자열 포함 여부

Java 주요 함수
equals()
contains()
startsWith()
endsWith()

---

### 길이
str.length()

---

### 문자 접근
str.charAt(i)

---

### 문자열 분리
str.split(" ")

---

### 부분 문자열
str.substring(a, b)

---

## 꿀팁

### 💡 문자열은 immutable

Java 문자열은 변경되지 않습니다.

따라서 변경이 많다면
StringBuilder

사용이 좋습니다.

---

### 💡 char vs String

문자 하나
'a'

문자열
"a"

---

### 💡 ASCII 활용
'a' → 97
'A' → 65

대소문자 변환 가능
c - 'a'

---

### 💡 문자열 문제는 대부분 O(N)

따라서 **단순 순회로 해결되는 경우가 많습니다.**
